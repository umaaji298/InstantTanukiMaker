[作成の流れ]
	画像追加からパーツ画像選んで一覧に追加。
	画像一覧から画像を選択してパーツの調整。
	合成プロパティで動きを確認後、画像を保存。


[パーツプロパティの変更]
	画像一覧からクリックしてパーツを選択することでプロパティの変更ができます。
	オフセット・回転はフレームごと、それ以外は画像全体に変更が適用されます。
	
	オフセット等の調整は直接数字を入力する以外にも
	プレビュー画像にマウスカーソルを合わせてクリック等でも操作できます。

	左右クリック:	フレームを前後に切り替え
	ホイール：	回転
	十字キー：	オフセット位置調整


[肌透過画像について]
	顔無し画像を追加すると一緒に画像一覧に追加されます。
	透過元の画像と重ねてその間にパーツを挟み込むことで
	パーツを髪の毛の後ろに配置したり、肌の色を変えることができます。
	
	おまけ機能からも肌透過画像を作成できますが、
	輪郭がはっきりしてかつ、うまるダンスと同じ肌の色の画像でなければ機能しません。
	誤検出が発生することもありますが、やっつけなので許してください。

[画像取込]
	取り込める画像はGIFとPNGだけです。
	画像一覧に顔無し画像がある場合は顔無し画像をベースに、
	顔なし画像がない場合は一番大きな画像、一番枚数の多い画像に合わせて
	画像サイズ、フレーム数が拡張されます。

[位置合わせ]
	動きがダンスに対応していない画像を扱う際は
	位置合わせにチェックを入れることで
	部位ごとに大まかな位置を合わせてくれます。
	細かい位置ずれはフレームごとにオフセット等を調整してください。


[不具合]
	GIF画像は256色しか扱えないのでグラデーションの効いたカラフルな画像は荒く表示されます。
	
	おまけ機能の肌透過画像追加には肌の未検出や誤検出があります。
	プリセットの肌透過画像もこの機能を利用して作ったのでいくつか誤検出のある画像があります。
